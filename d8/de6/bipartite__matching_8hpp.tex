\subsection{Detailed Description}
The class solve the bipartite weighted matching problem with the Hungarian method. 

The algorithm is based on the description available through \href{http://community.topcoder.com/tc?module=Static&d1=tutorials&d2=hungarianAlgorithm}{\tt http\+://community.\+topcoder.\+com/tc?module=\+Static\&d1=tutorials\&d2=hungarian\+Algorithm} in particular it has been implemented the O(n$^\wedge$3) version of the algorithm.
